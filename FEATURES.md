# Features

## Data schema

- [ ] User -> Posts (1:N)
- [ ] Post <-> Categories (N:M)
- [ ] Post -> Comments (1:N)
- [ ] User -> Comments (1:N)

## Auth

- [ ] Sign up (email and password)
- [ ] Log in (minimal jwt)
- [ ] Authorization Guard

## CRUD blogs

- [ ] Create blog (status: draft / published / archived)
- [ ] Update blog (details, category, status)
- [ ] List blogs (pagination, author filter, category filter, title filter )
- [ ] View blog details (author, category, comments)
- [ ] Remove blog

## Comments (1 level)

- [ ] Create comment in a blog (accept md)
- [ ] List blog comments (pagination)
- [ ] Remove comment from a blog

## Categories

- [ ] Base categories already seed in the system
- [ ] Create categories non existent
- [ ] List categories (pagination and search filter)
- [ ] Assign category to a blog
